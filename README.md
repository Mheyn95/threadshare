# threadSHARE

## Description 
Storefront where you can design and customize t-shirts, hoodies, sweatshirts, and jackets. For every $10 you spend, we donate $1 to charity. Come shop with us to help us reach our donation goals. 

## Built With
* CSS
* Javascript
* React
* Redux
* Express
* Mongoose
* GraphQL
* Stripe
